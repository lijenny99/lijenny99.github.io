Snapchat Website Redesign

Personal Project


In an effort to learn more about UI/UX design and front-end web development, I pursued a personal project of redesigning the website for Snapchat. I designed this website with desktop in mind, since most mobile users would likely directly visit the Apple App Store or Google Play Store to receive more information. Despite this, the website is still reponsive for all devices.

I began the project by sketching out some initial designs in my sketchbook. Inspiration for the initial landing page came from the Bootstrap element "jumbotron", which involves large text and a call to action button in front of a background image. Afterwards, the drawings were transfered to digital design. I used Sketch and Invision to prototype the screens of my project. This was my first time using both programs, and it proved to be a rewarding learning experience. The final protype can be viewed here, and images of the initial screen designs can be seen below.


Finally, I challenged myself to transfer my design into a website. Through this, I was able to expand my knowledge in HTML, CSS, Javascript, jQuery, and Bootstrap to build my first ever responsive website (the code for the final design can be viewed here). Although it's only a simple, one page site, it took me a few days to create, refine, and finally be satisfied with the final product. Through this project, I became more interested in front-end web development and design and will likely pursue more projects in the near future. Stay tuned.
